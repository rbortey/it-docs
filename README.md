# IT Documentation

Internal working notes for service desk, remote access, endpoint support, and routine operational procedures.

## Sections

- `runbooks/` - repeatable operational procedures
- `network/` - network and remote access notes
- `support/` - service desk handling notes
- `archive/` - old notes retained for reference

These documents are maintained for operational consistency. Anything sensitive should be issued through approved systems and rotated when an engagement ends.
